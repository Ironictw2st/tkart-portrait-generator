Place the trained LoRA checkpoint here as:

    output/v3/tk3k_portrait_v3.safetensors   (~436 MB)

Download from Hugging Face:

    https://huggingface.co/Ironictw2st/tk_3k_portraits
    (direct: https://huggingface.co/Ironictw2st/tk_3k_portraits/resolve/main/tk3k_portrait_v3.safetensors)

The app's LoRA dropdown lists every *.safetensors found in this folder, so you can
drop additional epoch checkpoints here too (e.g. tk3k_portrait_v3-000006.safetensors).

This file is intentionally NOT committed to git (too large). See the main README.
